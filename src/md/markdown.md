# 마크다운 문법   

## 마크다운이란 무엇인가   

특수기호와 문자를 이용한 매우 간단한 구조의 문법을 사용하여 웹에서도 보다 빠르게 컨텐츠를 작성하고 보다 직관적으로 인식할 수 있다.   마크다운이 최근 주목받기 시작한 이유는 깃허브 덕분이다. 깃허브의 저장소인 Repository의 정보를 기록하는 README.md 파일은 마크다운 문서로 이루어져있기 때문이다.   

## 마크다운의 장/단점   

### 장점   

- 간결하고 별도의 도구 없이 작성이 가능하다.   
- 텍스트로 저장되기 때문에 용량이 적어서 보관이 용이하다.   
- 지원하는 프로그램과 플랫폼이 다양하다.   

### 단점   

- 표준이 없고, 표준이 없기 때문에 도구에 따라서 변환방식이나 생성물이 다르다.   
- 모든 HTML 마크업을 대신하지 못한다.   

## 문법   

### 1. 헤더 (Headers)   

'#' 을 사용하여 작성하고, HTML의 'h1' 태그와 비슷하게 제목에서 사용할 수 있다.   
'#' 의 개수가 늘어갈수록 글자의 크기는 점점 감소한다.   
6개까지 밖에 지원하지 않는다.   

### 2. BlockQuote   

이메일에서 사용하는 '>' 블럭인용 문자를 이용한다.   

> 이 문장은 블럭인용 문장입니다.   

> > 안녕하세요.   

> > > - 안에 다른 마크다운 요소를 포함할 수 있다.   

### 3. 목록   

### 3-1 순서있는 목록   
숫자와 '.' 을 이용하여 목록을 만들 수 있다.   
1. 첫번째   
2. 두번째   
3. 세번째   

그러나 숫자의 위치를 바꿔도 순서는 내림차순으로 출력된다.   

1. 첫번째   
3. 세번째   
2. 두번째   

### 3-2 순서없는 목록   

"*", '+', '-' 을 사용하여 목록을 만들 수 있다.   
'*'을 사용하여 만든 목록   

* 빨강   
    * 노랑   
        * 파랑   

'+'을 사용하여 만든 목록   

+ 빨강   
    + 노랑   
        + 파랑   

'-'을 사용하여 만든 목록   

- 빨강   
    - 노랑   
        - 파랑   

여러가지를 혼합하여 사용하는 것도 가능하다.   

* 빨강   
    + 초록   
        - 파랑   
            * 노랑   

### 4. 코드   

### 4-1 들여쓰기   

4개의 공백이나 하나의 탭으로 들여쓰기가 된 문장을 만나게되면 변환되어 들여쓰기 되지 않은 행을 만날 때까지 변환이 된다.   

안녕하세요.   

    잘 부탁드립니다.   

감사합니다.   
두 번 째 줄만 들여쓰기가 된 것을 볼 수 있다.   

### 4-2 코드블럭   

코드블럭은 두가지 방식을 사용할 수 있다.   

- "<pre><code>{code}</code></pre>" 방식   

<pre>   
<code>   
안녕하세요.   
잘 부탁드립니다.   
감사합니다.   
</code>   
</pre>   

- 코드블럭 코드("```")를 이용하는 방식   

```   
안녕하세요.   
잘 부탁드립니다.   
감사합니다.   
```   

### 5. 수평선   

"* * *"   
"***"   
"*****"   
'- - -'   
'--------'   
위에 있는 예시들은 모두 수평선을 만든다.   

### 6. 링크   
링크를 거는 사용 문법으로   
* 외부링크   
"[Title](link)" 처럼 사용하고   
예시는 "[Google](https://google.com, "google link")" 이다.   

* 자동연결   
URL이나 이메일 주소인 경우   
URL: <https://example.com/>   
이메일링크: <address@example.com>   
와 같이 사용할 수 있다.   

### 7. 강조   
+ 하나의 '*'이나 '_'을 사용하여 강조할 수 있다.   
*안녕하세요*   
_안녕하세요_   

+ 두개의 '*'이나 '_'을 사용하여 강조할 수 있다.   
**안녕하세요**   
__안녕하세요__   

+ 두개의 '~'을 사용하여 취소선을 추가할 수 있다.   
~~안녕하세요~~   

### 8. 이미지   
링크 문법과 비슷하게 생겼지만 앞에 '!'가 붙는다.   
+ 인라인 이미지: '![text](이미지 파일 경로.jpg)'   
+ 링크 이미지: '![text](이미지 파일 URL)'   

이미지 파일에 마우스를 올렸을 때 커서 옆에 텍스트가 나오게 하려면   
'![text](이미지 파일 경로.jpg "이미지 이름")'   
'![text](이미지 파일 URL "이미지 이름")'   
과 같이 이미지 이름을 적어주면 된다.   

### 9. 줄바꿈   
문장이 끝나고 3칸 이상 띄어쓰기를 하면 문장이 바뀐다.   

+ 안녕하세요. (Enter만 입력하고 세칸 이상 띄우지 않음)
반갑습니다.   

+ 안녕하세요. (세칸 띄움)   
반갑습니다.
